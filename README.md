# chicago-aro
Chicago ARO Data

The chicago-aro dataset has been generated from Chicago's Dept of Planning & Development quarterly reports that detail the for sale and rental housing projects in the city that have triggered the Affordable Requirements Ordinance. Includes data from January 1, 2016 to September 30, 2018. Source documents are here: https://www.chicago.gov/city/en/depts/doh/supp_info/quarterly_housingreports.html
